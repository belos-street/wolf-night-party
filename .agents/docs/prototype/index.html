🤪 白痴：被投票可翻牌存活

【胜利条件】

好人胜利：所有狼人出局
狼人胜利：
• 5-9人局(屠城)：狼人数量≥好人
• 10-12人局(屠边)：屠神或屠民

夜晚 → 狼人刀人 → 预言家查验 → 女巫用药 → 守卫守护
白天 → 死亡通报 → 遗言 → 自由讨论 → 投票放逐
