# 人工联调测试手册（可勾选执行版）

> 适用范围：首版本地联机狼人杀（6-12 人）  
> 建议执行顺序：先跑 `P0` 全量，再跑 `P1` 角色/分支，最后跑 `12P` 增量专项

## 1. 使用说明

1. 该文档用于手工回归，目标是“可开局、可推进、可结算、可恢复”。
2. 每条用例执行后，在用例总表和用例详情里打勾。
3. 出现异常时，先记录缺陷，再继续后续不阻塞用例。
4. 若当前局角色不满足分支测试（如白痴/猎人），可快速重开新局补测。

## 2. 测试环境

- OS：Windows / macOS / Linux（任意）
- Node：`22.x`（最低 `22.13.0`）
- 包管理：`pnpm`
- 浏览器：Chrome / Edge（推荐）
- 项目根目录：`werewolf`

## 3. 启动步骤

1. 安装依赖（首次）：
```bash
pnpm install
```
2. 启动前后端：
```bash
pnpm dev
```
3. 打开前端地址（默认）：
`http://127.0.0.1:5173`
