# 狼人杀 - 本地联机版 胜利条件

## 1. 好人胜利
- 所有狼人出局

## 2. 狼人胜利
标准模式下（6-12人），狼人胜利条件由当前玩家人数决定：

### 2.1 屠城规则（6-9人局）
- 存活狼人数量 >= 存活好人数量时狼人获胜
- 规则简单，适合新手局

### 2.2 屠边规则（10-12人局）
- **屠神**：消灭所有神职角色（预言家、女巫、猎人、守卫、白痴）
- **屠民**：消灭所有平民
- 满足任一条件即胜利，策略深度更高

## 3. 胜利条件检测时机
系统在以下节点自动检测胜利条件：
1. 夜晚结算完成后（所有死亡结算完毕）
2. 猎人开枪后（猎人带走的玩家死亡后，包括狼刀触发和投票触发两种情况）
3. 女巫用药后（毒药生效，玩家死亡后）
4. 投票放逐后（被放逐玩家为白痴则翻牌并继续游戏；若为猎人则等开枪后再检测；其他角色死亡后检测）
5. 断线玩家被判死后（倒计时结束，玩家死亡后）

> 注意：每次检测胜利条件时，若游戏未结束，继续当前流程；若游戏结束，立即终止后续流程并宣布结果。

## 4. 各人数局胜利规则对照

| 总人数 | 狼人胜利规则 | 说明 |
|-------|------------|------|
| 6 | 屠城 | 狼人数量 >= 好人数量 |
| 7 | 屠城 | 狼人数量 >= 好人数量 |
| 8 | 屠城 | 狼人数量 >= 好人数量 |
| 9 | 屠城 | 狼人数量 >= 好人数量 |
| 10 | 屠边 | 屠神或屠民 |
| 11 | 屠边 | 屠神或屠民 |
| 12 | 屠边 | 屠神或屠民 |
