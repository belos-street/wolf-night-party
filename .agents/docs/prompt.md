我要实现一个本地线下联机的狼人杀游戏。
期望用手机的termux 运行后端服务，然后手机开放wifi热点，
游戏过程中，手机会作为服务器，其他设备通过局域网连接到手机，浏览器输入手机的ip地址+端口，即可加入游戏。

游戏内容：

- 游戏规则：标准狼人杀规则
- 游戏流程：标准狼人杀流程
- 游戏结束条件：标准狼人杀结束条件

玩家人数：5\~12人
阵营：狼人阵营和平民阵营
角色：请补充

服务端自动当上帝，作为状态机推动游戏流程。
所有玩家都在前端进行操作，服务端只负责处理前端的请求和响应。

技术栈：

- 后端：nodejs + typescript + fastify
- 前端：react + tailwindcss + shadcn-ui + 内置12个头像
- 服务器：termux
